# urufu
## description
[人狼Online](https://zinro.net)非公式クライアント。しかし中身はWebView2なのでセーフ。
## How To Build
1. [最新のHot Soup Processor 3.x](http://hsp.tv/idman/download.html)をインストールします。
2. HSPスクリプトエディタでmain.hspを開きます。
3. F5を押してビルドします
